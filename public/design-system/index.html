SECTION_01 // BRAND IDENTITY
S7 LABS

// The innovation sub-brand of System7.

Dark theme. Terminal/monospace voice. A single cyan-blue accent. No glassmorphism — instead, layered scanlines, dot grids, corner brackets, and breathing glows. Everything reads as a console: typewritten titles, mono uppercase labels with wide tracking, status pills with live dots.

SECTION_08 // ACCESSIBILITY & MOTION
  • prefers-reduced-motion — all animation declarations are gated; e.g. .orb, .caret, .voiceOrb, .back-link-arrow, .wordmark-arrow, and every t7-orb-* ring are zeroed under @media (prefers-reduced-motion: reduce).
  • Touch targets — buttons in mobile (<640px) bumped to min 36×36px (icon-btn, mode-btn, back-link, li-btn). Send button bumped to 40×40px.
  • iOS auto-zoom — the chat .field uses 16px font on mobile to suppress Safari focus-zoom; other mono UI stays at desktop sizes.
  • Body scroll lock — the S7ChatWidget locks body { overflow: hidden } while expanded; root container expands to full viewport on mobile (top/right/bottom/left:8px).
  • Mobile breakpoints — 760px (layout reflow) and 640px (mobile-first sizes, sticky CTA stacks, hidden labels).
  • Header backdrop — uses backdrop-filter: blur(16px) saturate(140%); falls back to a solid color-mix background.
